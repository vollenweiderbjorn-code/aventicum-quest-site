Le paiement est-il sécurisé ?

Oui. Stripe est l'un des leaders mondiaux du paiement en ligne. Aucune donnée bancaire ne transite par nos serveurs.

Que faire si je ne reçois pas mon code ?

Vérifiez vos spams. Si le code n'arrive pas sous 5 minutes, écrivez à contact@aventicum-quest.ch avec votre confirmation de paiement.

Un seul achat suffit pour tout mon groupe ?

Absolument. Un seul code par groupe, pas besoin que chaque participant achète séparément. Le code se joue sur un téléphone partagé ou plusieurs appareils.
